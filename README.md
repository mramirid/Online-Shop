## How to run?

1. Create a Mongo Atlas cluster & connect this app to your cluster (fill the .env)

2. You need to configure your own mailgun (fill the .env)

2. Run npm install

```bash 
$ npm install
```

3. Compile the TypeScript sources

```bash 
$ tsc --watch
```

4. Run nodemon using my configuration

```bash 
$ npm start
```
